Optional onPreCall

onPreCall?: ((ctx, request) => Promise<void>)

Method which is executed before an actual call to an RPC method. Pre-call checks should execute all necessary checks (such as authentication, authorization, throttling, etc.) before allowing the real method to proceed. Pre-call checks should throw, if for any reason the call should not proceed. Return void to allow execution of the actual call.
