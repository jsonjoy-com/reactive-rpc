Class RpcMessageBatchProcessor<Ctx>

A global instance for processing RPC messages. Used for request/response like scenarios. For streaming scenarios, use RpcMessageStreamProcessor.

This processor can be shared across different connection/requests as "ctx" is passed on each call and not state is held.
